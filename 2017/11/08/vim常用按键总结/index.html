全局配置

  • 单次启用某个配置项，在命令模式下，先输入一个冒号，再输入配置
  • 用户个人的配置在~/.vimrc
  • Vim 的全局配置一般在/etc/vim/vimrc或者/etc/vimrc，对所有用户生效
